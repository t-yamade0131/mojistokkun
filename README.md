# もじすとっくん

クリップボード履歴の管理と、よく使うテキストのお気に入り登録ができる Chrome 拡張機能です。

---

## できること

### 履歴タブ
- コピー（Ctrl+C）・カット（Ctrl+X）したテキストを自動で最大 **20件** 記録
- 一覧から選んでクリックするだけで再コピー
- ☆ボタンでお気に入りに登録。★になったものは誤削除を防ぐため削除ボタンが非表示に
- テキスト検索でフィルタリング

### お気に入りタブ
- 最大 **30件** のテキストを名前付きで登録・管理
- プレビューパネルで全文確認 → コピー

### お気に入り管理（⚙ボタン）
- タイトル・本文の新規追加・編集・削除

### キーボード操作
| キー | 動作 |
|------|------|
| `Alt+V` | Popup を開く（Chrome のショートカット設定で変更可） |
| `↑` / `↓` | リスト選択 |
| `←` / `→` | アクション切替（コピー／☆／削除） |
| `Tab` | 履歴タブ ↔ お気に入りタブ 切替 |
| `Ctrl+F` | 検索バーにフォーカス |
| `Enter` | 選択中のアクションを実行 |
| `Esc` | 一段階戻る（Popup を閉じる） |

## インストール方法

1. このリポジトリを ZIP ダウンロード または `git clone`
2. Chrome で `chrome://extensions` を開く
3. 右上の「**デベロッパーモード**」をオン
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
5. ダウンロードした中の **`mojistokkun` フォルダ**を選択

> **注意:** 拡張機能をインストールした後に開いていたタブは、一度リロード（F5）しないとコピー履歴が記録されません。

## 裏側のしくみ（技術補足）

Chrome 拡張機能の **Manifest V3** という最新仕様で作られています。

- **コピー検知:** ページに常駐する小さなスクリプト（Content Script）が `copy` / `cut` イベントを監視し、テキストを取得しています。パスワード入力欄は除外しています。
- **データ保存:** Chrome が拡張機能向けに提供するローカルストレージ（`chrome.storage.local`）に履歴・お気に入りを保存します。ブラウザの外には一切送信しません。
- **Popup UI:** ツールバーアイコンをクリックすると開く小ウィンドウ（Popup）で、HTML / CSS / JavaScript のみで構成されています。フレームワーク・ビルドツールは使っていないシンプルな構成です。
